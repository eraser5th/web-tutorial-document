# レイアウト

Webページには典型的なレイアウトがいくつかあります

- ヘッダー
- フッター
- サイドバー
- など

いい感じのWebページを作る際はこれらが骨組みになります

実際にHTMLにしてみるとこんな感じです

```html
<!DOCTYPE HTML>

<html>
  <head>
  </head>

  <body>
    <header>
    </header>

    <main>
    </main>

    <footer>
    </footer>
  </body>
</html>
```

YouTubeやTwitterなど、身近なサイトのレイアウトを見てみましょう！
