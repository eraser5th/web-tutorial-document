# CSSを書くにあたって

CSSを書く際、基本的に`class`に対して見た目を変更していくことが多いです  
(なぜ`id`ではないのか、というのは優先度の関係などによるものです)

例.

```html
<div class="big-text">hello!</div>
```

```css
.big-text {
  font-size: 5000px;
}
```

また、次のように階層構造にして書くこともあります

```html
<div class="item">
  <div class="item-text">aiueo</div>
  <button class="item-button">これはボタン</button>
</div>
```

```css
.item {
  background-color: #eeeeee;
}

.item-text {
  font-size: 50px;
  color: #080808;
}

.item-button {
  background-color: #ff0000;
  color: #ffffff;
}
```
