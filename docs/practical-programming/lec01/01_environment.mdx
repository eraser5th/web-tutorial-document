## エディタ編

エディタとはファイルを編集するツールのことを指します  
テキストエディタ、統合開発環境(IDE)みたいにいくつか種類があります

この実践的プログラミングでは使いやすさ&個人的な好みから[Visual Studio Code](https://code.visualstudio.com/)(VS Code)を紹介します  
VS Codeのインストールは以下のページから行ってください

- [公式ダウンロードページ](https://code.visualstudio.com/download)
- [ダウンロードに関する補足資料](https://hackmd.io/@wX9YOe1-SXq2KK1SjrAELg/ry9mgljLc)

<details><summary>よく使われがちな(主観)エディタ一覧</summary>

- Vim
  - キーバインディングが特殊で、かなり人気
- Emacs
  - これも人気。「Vim vs Emacs」とかって調べると宗教戦争を見られる
- Neovim
  - Vimの派生。人気
- nano
  - 操作方法がわかりやすい
- Visual Studio Code
  - かなーーり人気。汎用性の塊
- Visual Studio
  - C#とかMicroSoft系との相性が良い。あとゲーム開発とかでも使われるイメージ
- IntelliJ
  - JVM系を扱うんだったらこれが使われがち。機能が豊富

他にも色々あるのでぜひ好きなエディタを使ってみてね！

ちなみに、基本的にエンジニアは好きなエディタを使ってプログラミングをし、使うエディタをどれにするかは個人の宗教によります  
道行く人に「好きなエディターはなんですか？」とかって聞くと高確率で熱弁されたりします

</details>
