# 再帰関数 (Recursive function)

関数が自分自身を内部で呼び出す関数を **再帰関数** と呼ぶ。  
基本的に受け取った引数を少しずつ変えながら次の自分を呼び出す。

数学の漸化式だと思ってください。

例えば以下の漸化式は、

$$
\begin{split}
a_0 &= 0\\
a_n &= 2 + a_{n - 1}
\end{split}
$$

以下のプログラムで表せる。

```javascript
function a(n) {
  if (n === 0) {
    return 0
  } else {
    return 2 + a(n - 1)
  }
}
```
