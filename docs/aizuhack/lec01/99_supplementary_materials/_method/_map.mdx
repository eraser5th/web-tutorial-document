### Array.map()

配列の各値に対し関数を適応した値の配列を返します。（日本語むずい😅）

:::info
めちゃめちゃ便利なので余裕のある方は使って慣れていくといいです。
:::

例） 配列の全要素を2倍する

```javascript
const a = [1, 2, 3]
const mappedA = a.map((v) => {
  return v * 2
})

console.log(mappedA)

// aは変わらない
console.log(a)
```

詳細は[こちら](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference/Global_Objects/Array/map)
