# `while`文(繰り返し)

書き方

```javascript
while (継続条件) {
  // 処理内容
}
```

while文は以下の順に実行される

1. `継続条件`。継続条件の評価が`true`であればこのまま続き、`false`の時は`while文`が終了する。
2. `処理内容`
3. 1に戻る

## `break`

`for文`と同じく`while文`の中で`break`を実行すると、`while文`が終了する。

```javascript
let i = 0
while (i < 100) {
  if (n === 50) {
    break
  }
  console.log(i)
  i = i + 1
}
```

## `continue`

`for文`と同じく`while文`の中で`continue`を実行すると、その回のループが終了し次のループに入る。

```javascript
let i = 0
while (i < 100) {
  if (i % 2 === 1) {
    continue
  }
  console.log(i + "は偶数です！")
  i = i + 1
}
```
