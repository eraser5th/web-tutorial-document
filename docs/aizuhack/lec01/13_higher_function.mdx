# 高階関数

引数に関数を受け取る関数、または関数を返り値として返す関数を **高階関数** と呼ぶ。

JavaScriptにおいて、関数は数値や文字列、オブジェクトなどと同じく値です(実際に関数はオブジェクトの一種として実現されている)。
従って、returnを使って関数を返したり、引数に関数を受け取ることができます。

特に前者の引数に関数を受け取る関数は頻出なため理解しておくと良い。
