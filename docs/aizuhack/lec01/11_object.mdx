また、オブジェクトにはメソッドと呼ばれる関数を含めることができる。  
（関数についてはこの次の章でやるのでなんとなくでいいです）

```javascript
let myProfile = {
  lastName: "Kato",
  firstName: "Takeshi",
  birthday: "09/13",
  getFullName() { // これがメソッド
    return this.lastName + " " + this.firstName;
  }
}

console.log(myProfile.getFullName())
```
