# プロパティ

CSSには様々なプロパティ(200種類ぐらい？)があり、それぞれにそれぞれ特有の値を入れることができる。

一つ一つやっていると終わらないので
[このサイト](https://www.w3schools.com/cssref/index.php)や普通に検索しながらやってみてほしい。

なくても一応見た目がカッコよくないだけで動かなくはないので、カッコよくしたいときに調べると良い。  
「こんな見た目を作りたい」と思ったら、「CSS 背景」などのように検索すれば割とやりかたが出てきます。

例

```css
h1 {
  color: lightblue;
  font-size: 48px;
}
```
