# セレクター

セレクターは、いろいろな種類がある。

- タグ
- class
- id
- 擬似要素
- 擬似クラス

また、これらのセレクタを組み合わせることにより複雑な選択が可能。（これについては触れないでおく）

## タグセレクタ

HTMLのタグを選択する

```css
h1 {
}
p {
}
```

## classセレクタ

HTMLのclass属性を見て選択することができる。  
クラス名の前に`.`をつければ良い。

```css
/* hoge というクラスを選択する場合 */
.hoge {
}
```

## idセレクタ

```css
/* hoge というidを選択する場合 */
#hoge {
}
```
