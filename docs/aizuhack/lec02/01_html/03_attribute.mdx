# 属性(attribute)

HTMLの要素にはそれぞれ属性を持たせることができる。

例えば

- `img`タグの`src`属性に画像のURLを持たせることによりその画像を表示することができる
- `a`タグの`href`属性にURLを持たせることによりリンクを作成できる

要素によって使う属性は異なるため覚えるのは大変。  
適宜調べるのが良いでしょう。

## 使い方

属性の指定方法は、HTMLのタグの中に`属性名="値"`と書き込めば良い。
