# JavaScriptでHTTP通信を行う

JavaScriptでHTTP通信は`fetch`という関数を使う。
使い方としては、第一引数に`URL`を文字列で、第二引数にメソッドやヘッダー、ボディなどが入ったオブジェクトを渡す。

```javascript=
fetch('URL', {　ここに色々な付加情報を加える })

// 付加情報が何もない時は第二引数はなくても良い
fetch('URL')
```

また、`fetch` 関数は非同期関数であるため返り値は `Promise` となります。
なので今回は先ほどと同じように呼び出す際には非同期関数の中で `await` をつけて実行しましょう。
