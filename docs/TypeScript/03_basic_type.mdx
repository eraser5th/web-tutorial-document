# 基本的な型

## number型

全ての数値のみを要素とする型です。

```typescript
const a: number = 1
const b: number = 3.1415
const c: number = -100
const d: number = Infinity
const d: number = NaN
```

## string型

全ての文字列のみを要素とする型です。

```typescript
const a: string = "hoge"
const b: string = ""
const c: string = "1"
const d: string = "-100"
const d: string = "{ hoge: 'aaa' }"
```

## boolean型

真偽値である`true`と`false`のみを要素とする型です。

```typescript
const a: boolean = true
const b: boolean = false
```

## undefined型

`undefined`のみを要素とする型です。

```typescript
const a: undefined = undefined
```

## null型

`null`のみを要素とする型です。

```typescript
const a: null = null
```
