## nvmのインストール 

WSL上で以下を実行。

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```

ターミナルを再起動した後に以下を実行してください

```sh
nvm -v
```

このような表示が出ていればnvmのインストールは完了です。(数値は多少異なっても大丈夫です)

```sh
0.39.3
```

## nodeのインストール

ターミナルで以下を実行。

```sh
nvm install v18.16.0
```

以下を実行してインストール出来たか確認してください。

```sh
node -v
```

このような表示が出ていればNode.jsのインストールは完了です。

```sh
v18.16.0
```
