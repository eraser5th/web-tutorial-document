## nvmのインストール 

WSL上で以下を実行。

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.3/install.sh | bash
```

```sh
source ~/.bashrc
```

以下を実行してインストール出来たか確認してください。

```sh
nvm -v
```


このような表示が出ていればOKです。(数値は多少異なっても大丈夫です)

```sh
0.39.3
```

## nodeのインストール

ターミナルで以下を実行。

```sh
nvm install v18.16.0
```

以下を実行してインストール出来たか確認してください。

```sh
node -v
```

このような表示が出ていればOKです。

```sh
v18.16.0
```
